## MySQL 笔记

*连接数据库*

```sql
mysql -uroot -p
```

*退出数据库*

```sql
//注意要添加分号
exit;
```

*查看数据库*

```sql
SHOW DATABASES;
```

*创建数据库*

```sql
CREATE DATABASE foxebook;
```

*进入数据库*

```sql
USE foxebook
```

*查看数据库中的表*

```sql
SHOW TABLES
```
